### [DirtyClone Linux 内核本地提权漏洞可获取 root 权限](https://research.jfrog.com/post/dissecting-and-exploiting-linux-lpe-variant-dirtyclone-cve-2026-43503/) ⭐️ 9.0/10 [技术]

JFrog 安全研究团队披露了编号为 CVE-2026-43503、名为 DirtyClone 的 Linux 内核本地提权漏洞，CVSS 评分为 8.8。该漏洞允许非特权本地用户通过利用 __pskb_copy_fclone() 在 IPsec 数据包处理过程中丢失 SKBFL_SHARED_FRAG 标志的问题，将权限提升至 root。 该漏洞对主要 Linux 发行版、云环境和 Kubernetes 集群构成严重风险，尤其是那些默认启用非特权用户命名空间的系统。成功利用后可静默篡改特权可执行文件且不留下内核日志，使得检测变得困难。 该漏洞是 DirtyFrag 家族的新变种，由 __pskb_copy_fclone() 及相关函数在克隆 socket buffer 时未能传播 SKBFL_SHARED_FRAG 标志导致。修复已于 5 月 21 日在 Linux v7.1-rc5 中发布，Ubuntu 等发行版已提供补丁内核。

telegram · zaihuapd · 6月27日 08:00

**背景**: Linux 内核使用 socket buffer (sk_buff) 来管理网络数据包数据，其中共享片段标志 (SKBFL_SHARED_FRAG) 表示内存是只读的并在多个缓冲区之间共享。当该标志在克隆过程中丢失时，内核可能将只读的 page cache 内存误判为可写，从而允许攻击者修改特权文件。DirtyClone 继承了 Dirty COW 和 Dirty Pipe 等先前 Linux 内核本地提权漏洞的传统，这些漏洞都涉及内存处理缺陷。

### [IP Crawl：公开网络摄像头的活地图引发隐私担忧](https://ipcrawl.com/) ⭐️ 8.0/10 [技术]

一个名为 IP Crawl 的网站聚合了公开互联网上可访问的数千个开放网络摄像头的活地图，在所有者不知情的情况下暴露了私人和公共空间的实时画面。 这突显了一个持续存在的安全和隐私漏洞：许多消费级 IP 摄像头未加保护地暴露在互联网上，可能让任何人窥视家庭、企业和私人时刻。 该网站索引了仍使用默认密码或无需认证的摄像头，通常来自廉价的中国制造设备；这个问题至少自 2012 年以来就已为人所知，但仍然普遍存在。

hackernews · arm32 · 6月27日 19:09 · [社区讨论](https://news.ycombinator.com/item?id=48700834)

**背景**: 许多消费级 IP 摄像头设计为可通过互联网远程访问，但用户往往不更改默认密码或启用安全功能。这使得任何知道摄像头 IP 地址的人都能查看实时画面。像 IP Crawl 这样的网站扫描互联网寻找此类设备并聚合它们，创建了一个可搜索的不安全摄像头地图。

**社区讨论**: 评论者对隐私侵犯表示不安，一些人指出大多数用户只是按照说明操作，并不了解安全风险。其他人指出这个问题已经存在十多年且仍未改变，少数人则推测了潜在的合法用途，如盗窃检测或长期存档。

一篇《今日心理学》文章和 Hacker News 上的讨论探讨了像 Ozempic 这样的 GLP-1 激动剂如何改变肠脑轴，用户报告称食欲下降、情绪改善以及成瘾行为发生变化。 这很重要，因为它表明 GLP-1 激动剂可能具有超越减肥的益处，可能有助于心理健康和成瘾治疗，但也引发了关于停药后体重反弹以及需要终身用药的担忧。 肠脑轴是连接胃肠道和中枢神经系统的双向通信网络；GLP-1 受体存在于胰腺β细胞和脑神经元上。社区评论描述了“食物噪音”减少、睡眠改善，以及恶心和腹部压痛等副作用。

hackernews · randycupertino · 6月27日 21:34 · [社区讨论](https://news.ycombinator.com/item?id=48701984)

**背景**: GLP-1 受体激动剂（如 semaglutide，即 Ozempic）模拟肠促胰岛素激素 GLP-1，刺激胰岛素分泌、延缓胃排空并促进饱腹感。肠脑轴涉及神经、激素和免疫通路，使肠道微生物组和消化系统能够影响大脑功能。这些药物主要用于 2 型糖尿病和肥胖症，但新兴研究表明它们对情绪、成瘾和炎症也有影响。

### [检查抖音自动扣费：汽水音乐已扣费一年多](https://www.v2ex.com/t/1223379#reply5) ⭐️ 7.0/10 [社会热点]

一位用户发现抖音旗下的汽水音乐应用在未经其明确同意的情况下，默默自动续费 SVIP 会员超过一年，造成持续扣费。该用户提醒其他人检查自己的账户是否存在类似未经授权的扣费。 这一事件凸显了抖音等热门平台上自动续费功能存在的普遍消费者权益问题，可能影响数百万用户。欺骗性的通知方式和大量的投诉表明，需要更严格的监管和提高用户意识。 该用户在 2025 年初以 1 分钱开通了 7 天试用，随后该试用在未明确通知的情况下自动转为付费 SVIP 会员。自动续费通知仅在汽水音乐应用内发送，极易被忽略。该用户还指出，关于汽水音乐自动续费的投诉量超过了其他几家音乐平台的总和。

**背景**: 许多应用和在线服务提供免费或低价试用，试用期结束后会自动转为付费订阅，除非用户在试用结束前手动取消。这种被称为自动续费的做法，往往依赖于小字条款和隐蔽的通知设置，导致用户产生意外扣费，且可能长期累积。部分地区消费者保护法要求明确披露和便捷的取消选项，但执行力度不一。
